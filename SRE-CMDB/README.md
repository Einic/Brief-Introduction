## 本项目不开源

1、支持自动化资产管理；  
2、支持自动化持续集成；  
3、支持工单数据库审核；  
4、支持自动化监控预警；  
5、支持安全堡垒机；  
6、支持自动化安全扫描；  
7、支持双因子或OpenLDAP登录;  
8、基于RBAC权限管理设计；
